# 工作台模板导出 · 施工指令

## 目标

把当前工作台配置（上下文区卡片、工具区卡片、模式卡、模型和设置）打包成一张 .card。别人导入后，所有卡片出现在画布上，README 告诉你怎么配置成一样的工作台。

## 做法

Workbench 加一个按钮"导出模板"。点击 → 收集当前配置 → 打包 → 下载。

---

## 一、收集配置

```typescript
const exportTemplate = useCallback(async () => {
  // 1. 收集上下文区 + 工具区 + 模式卡的 cardId
  const contextCardIds = contextOrder
    .filter((item) => item.type === "card")
    .map((item) => item.ref);
  const toolCardIds = toolItems.map((t) => t.cardId);
  const allExportIds = new Set([...contextCardIds, ...toolCardIds]);
  if (modeCardId) allExportIds.add(modeCardId);

  if (allExportIds.size === 0) {
    // Toast: "没有可导出的内容"
    return;
  }

  // 2. 打包每张卡
  const idx = getIndex();
  if (!idx) return;

  // 生成配置说明 README
  const readmeLines = [
    `# ${card.label || "工作台模板"}`,
    "",
    `> 导出时间：${new Date().toLocaleString()}`,
    "",
    "## 上下文卡片",
    ...contextCardIds.map((cid) => {
      const c = allCards.find((x) => x.id === cid);
      return c ? `- ${c.label}` : `- (未找到)`;
    }),
    "",
    "## 工具卡片",
    ...toolCardIds.map((cid) => {
      const c = allCards.find((x) => x.id === cid);
      return c ? `- ${c.label}${c.hasScripts ? " (可执行)" : ""}` : `- (未找到)`;
    }),
    "",
    "## 使用方式",
    "",
    "1. 导入此 .card 到 Smelt 空间",
    "2. 将卡片拖入工作台的上下文区和工具区",
    modeCardId ? "3. 将模式卡拖入工作模式凹槽" : "",
  ].filter(Boolean);

  const readmeBlob = new Blob([readmeLines.join("\n")], { type: "text/markdown" });
  const readmeFile = new File([readmeBlob], "README.md");

  // 打包第一张卡（作为 .card 的主卡），把其他卡的内容放到 extraFiles
  const mainCardId = allExportIds.values().next().value;
  const mainCard = allCards.find((c) => c.id === mainCardId);
  if (!mainCard || !mainCard.sphereId) return;

  const blob = await buildCardPackage({
    card: mainCard,
    sphere: idx.spheres[mainCard.sphereId],
    shell: BUILTIN_SHELLS[0].theme,
    extraFiles: [{
      file: readmeFile,
      name: "README.md",
    }],
    manifest: {
      id: crypto.randomUUID?.(),
      label: card.label || "工作台模板",
      version: "1.0.0",
      author: "Smelt User",
      description: `上下文 ${contextCardIds.length} 张 · 工具 ${toolCardIds.length} 张` +
        (modeCardData ? ` · 模式：${modeCardData.type}` : ""),
      tags: ["工作台模板"],
      requires: [],
      createdAt: Date.now(),
      exportedAt: Date.now(),
    },
    password: null,
    authorName: "Smelt User",
  });

  downloadBlob(blob, `${card.label || "workbench-template"}.card`);
}, [contextOrder, toolItems, modeCardId, modeCardData, allCards, card]);
```

## 二、UI 入口

在 Workbench 底部按钮区加一个按钮：

```typescript
<button onClick={exportTemplate} style={btnSm}>
  {t("wb.exportTemplate")}
</button>
```

就放在 `[归档]` 和 `[清除]` 旁边。

## 三、仅导出当前工作台引用的卡片

当前实现只导出第一张卡的内容。需要把上下文区和工具区的所有卡片内容都放入 .card。

改成：遍历 `allExportIds`，每张卡的球内容读出来→全部放 extraFiles：

```typescript
// 收集所有导出卡片的内容
const extraFiles: ExtraFile[] = [{
  file: readmeFile, name: "README.md",
}];

for (const cid of allExportIds) {
  if (cid === mainCardId) continue; // 主卡已在 sphere 里
  const c = allCards.find((x) => x.id === cid);
  if (!c?.sphereId) continue;
  try {
    const files = await readSphereFiles(c.sphereId);
    for (const f of files) {
      if (f.name === "_snapshot.png") continue;
      extraFiles.push({
        file: new File([new Blob([f.content])], f.name),
        name: f.name,
      });
    }
  } catch { /* skip */ }
}
```

## 五、验收

1. `npx tsc --noEmit` 零错误
2. 工作台有上下文 2 张卡 + 工具 1 张卡 + 模式卡 → 点"导出模板" → 下载 .card
3. 拖下载的 .card 到画布 → 导入面板 → 文件列表里包含 README.md + 上下文卡片 + 工具卡片
4. README.md 里列出了上下文卡片名、工具卡片名、使用方式
5. 空工作台（无上下文无工具）→ 点"导出模板" → 提示"没有可导出的内容"
