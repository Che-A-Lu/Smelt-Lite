# 多选与框选 · 施工指令

## 现状

- 单选：点击卡片 → `selectedId`（单个 state）→ 边框 2px + 发光环
- 点击空白 → 取消
- Delete → 删一张

## 目标

- Shift+单击 → 追加选中（不取消已有）
- Ctrl/Cmd+单击 → 切换选中
- 空白区 Shift+拖拽 → 画矩形框 → 框内卡片全选
- 多张选中时拖任意一张 → 全部跟随移动（保持相对位置）
- 多张选中时 Delete → 批量删除（带确认）
- 右键菜单 → "导出选中"（单张="导出"，多张="导出选中 N 张"）

```typescript

// 新：
```

```typescript
const boxSelect = useRef({
```

### 1.3 卡片点击改成支持修饰键

```typescript
const onCardClick = useCallback((cardId: string, e?: React.MouseEvent) => {
  const shift = e?.shiftKey ?? false;
  const ctrl = e?.metaKey || e?.ctrlKey;

  if (shift) {
    // 追加选中
    setSelectedIds((prev) => {
      const next = new Set(prev);
      if (next.has(cardId)) return prev;
      next.add(cardId);
      return next;
    });
  } else if (ctrl) {
    // 切换选中
    setSelectedIds((prev) => {
      const next = new Set(prev);
      if (next.has(cardId)) next.delete(cardId);
      else next.add(cardId);
      return next;
    });
  } else {
    // 普通点击 → 单选
    setSelectedIds(new Set([cardId]));
  }
}, []);
```

### 1.4 框选逻辑

`onPointerDown` 中加 Shift 检测。Shift+空白区拖拽 → 走框选，不走平移。

```typescript
const onPointerDown = useCallback((e: React.PointerEvent) => {
  if (e.target !== contentRef.current && e.target !== containerRef.current) return;

  if (e.shiftKey) {
    // 框选模式
    const rect = containerRef.current?.getBoundingClientRect();
    if (!rect) return;
    const v = view.current;
    const bx = (e.clientX - rect.left - v.x) / v.zoom;
    const by = (e.clientY - rect.top - v.y) / v.zoom;
    boxSelect.current = { active: true, startX: bx, startY: by, currentX: bx, currentY: by };
    setBoxRect({ x: bx, y: by, w: 0, h: 0 });
    (e.target as HTMLElement).setPointerCapture(e.pointerId);
    return; // ← 不触发平移
  }

  // 否则：点击空白取消选中 + 开始平移（原逻辑不变）
  setSelectedIds(new Set());
  cancelAnimationFrame(animFrame.current);
  const p = pan.current;
  p.active = true;
  // ... 省略原有平移逻辑，不变
}, [selectedIds]);
```

`onPointerMove` 中加框选更新：

```typescript
const onPointerMove = useCallback((e: React.PointerEvent) => {
  // 框选中 → 更新矩形
  if (boxSelect.current.active) {
    const rect = containerRef.current?.getBoundingClientRect();
    if (!rect) return;
    const v = view.current;
    const bx = (e.clientX - rect.left - v.x) / v.zoom;
    const by = (e.clientY - rect.top - v.y) / v.zoom;
    boxSelect.current.currentX = bx;
    boxSelect.current.currentY = by;
    const x = Math.min(boxSelect.current.startX, bx);
    const y = Math.min(boxSelect.current.startY, by);
    const w = Math.abs(bx - boxSelect.current.startX);
    const h = Math.abs(by - boxSelect.current.startY);
    setBoxRect({ x, y, w, h });
    return; // ← 制止平移
  }

  // 平移中（原逻辑不变）
  // ...
}, []);
```

`onPointerUp` 中完成框选判定：

```typescript
const onPointerUp = useCallback(() => {
  // 框选结束 → 找框内卡片
  if (boxSelect.current.active) {
    boxSelect.current.active = false;
    if (boxRect && boxRect.w > 10 && boxRect.h > 10) {
      // 最小 10px 才算框选（防止误触）
      const inBox = cards.filter((c) => {
        const cx = c.position.x;
        const cy = c.position.y;
        return cx >= boxRect.x && cx <= boxRect.x + boxRect.w &&
               cy >= boxRect.y && cy <= boxRect.y + boxRect.h;
      });
      if (inBox.length > 0) {
        setSelectedIds(new Set(inBox.map((c) => c.id)));
      }
    }
    setBoxRect(null);
    return;
  }

  // 平移结束（原逻辑不变）
  // ...
}, [cards, boxRect]);
```

### 1.5 多卡拖拽

在 `onCardDragMove` 中检测：如果拖拽的卡片在选中集合中，所有选中的卡片一起移动：

```typescript
const onCardDragMove = useCallback((cardId: string, cx: number, cy: number, px: number, py: number) => {
  // 如果拖的是多选中的一张 → 所有选中卡一起移动
  if (selectedIds.has(cardId) && selectedIds.size > 1) {
    const draggedCard = cards.find((c) => c.id === cardId);
    if (draggedCard) {
      const oldCx = dsRefs.current.get(cardId)?.oldCx ?? draggedCard.position.x;
      const oldCy = dsRefs.current.get(cardId)?.oldCy ?? draggedCard.position.y;
      const dx = cx - oldCx;
      const dy = cy - oldCy;
      for (const id of selectedIds) {
        const c = cards.find((x) => x.id === id);
        if (c && id !== cardId) {
          // 通过 DOM 直接更新其他卡片位置
          const el = cardElements.current.get(id);
          if (el) {
            el.style.transform = `translate(${c.position.x + dx}px, ${c.position.y + dy}px)`;
          }
        }
      }
    }
  }
  // ... 原有单个拖拽视觉反馈逻辑
}, [cards, selectedIds]);
```

这里有个微妙点——需要维护每张卡片的"本次拖拽起始位置"。最简单的做法是在 `onCardDragStart` 时记录所有选中卡片的位置：

```typescript
const onCardDragStart = useCallback((cardId: string) => {
  // 记录所有选中卡的起始位置（用于多卡拖拽偏移计算）
  if (selectedIds.has(cardId) && selectedIds.size > 1) {
    for (const id of selectedIds) {
      const c = cards.find((x) => x.id === id);
      if (c) multiDragOrigins.current.set(id, { x: c.position.x, y: c.position.y });
    }
  }
}, [cards, selectedIds]);
```

`onCardDragEnd` 时对每张选中卡调 `updateCard`：

```typescript
const onCardDragEnd = useCallback((cardId: string, cx: number, cy: number) => {
  if (selectedIds.has(cardId) && selectedIds.size > 1) {
    const orig = multiDragOrigins.current.get(cardId);
    const dx = orig ? cx - orig.x : 0;
    const dy = orig ? cy - orig.y : 0;
    for (const id of selectedIds) {
      const origPos = multiDragOrigins.current.get(id);
      if (origPos) {
        updateCard(id, { position: { x: origPos.x + dx, y: origPos.y + dy } });
      }
    }
    multiDragOrigins.current.clear();
  } else {
    // 单个拖拽落点判定（原逻辑）
    // ...
  }
}, [cards, selectedIds]);
```

### 1.6 批量删除

Delete 键处理改为：

```typescript
if (e.code === "Delete" || e.code === "Backspace" && selectedIds.size > 0 && !deletingId) {
  // 多张：弹确认
  if (selectedIds.size > 1) {
    if (window.confirm(`删除 ${selectedIds.size} 张卡片？`)) {
      for (const id of selectedIds) deleteCard(id);
      setSelectedIds(new Set());
      onRefresh();
    }
  } else {
    // 单张：直接删（带动画）
    setDeletingId([...selectedIds][0]);
    setSelectedIds(new Set());
  }
}
```

## 三、框选矩形 UI

Canvas 渲染一个蓝色半透明矩形（当 `boxRect` 不为 null 时）：

```typescript
{boxRect && (
  <div style={{
    position: "absolute",
    left: boxRect.x, top: boxRect.y,
    width: boxRect.w, height: boxRect.h,
    border: "1px solid #3b82f6",
    background: "rgba(59, 130, 246, 0.08)",
    pointerEvents: "none", zIndex: 9999,
  }} />
)}
```

这个矩形在 contentRef 内部，不带 transform（boxRect 的坐标已经是画布坐标）。

如果画布有缩放和平移，坐标需要变换。将 boxRect 放在 contentRef 内：

```typescript
{boxRect && (
  <div style={{
    position: "absolute",
    left: `${boxRect.x}px`, top: `${boxRect.y}px`,
    width: `${boxRect.w}px`, height: `${boxRect.h}px`,
    border: "1px solid #3b82f6",
    background: "rgba(59, 130, 246, 0.08)",
    pointerEvents: "none", zIndex: 9999,
  }} />
)}
```

因为这已经在 contentRef 内，position 自动跟随画布。不需要额外变换。
