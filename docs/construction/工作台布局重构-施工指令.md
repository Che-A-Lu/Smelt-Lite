### Drawer 组件

```typescript
function Drawer({ zoneId, anchorEl, zoneProps, onClose }) {
  // 获取被点图标的屏幕 Y 坐标
  const rect = anchorEl?.getBoundingClientRect();
  const top = rect ? rect.top : 200;

  return (
    <div style={{
      position: "fixed",
      left: /* 工作台浮窗 left - 250 */,
      top: top,
      width: 250, maxHeight: 300,
      background: "#fff", border: "1px solid #d1d5db",
      zIndex: 200, overflow: "auto",
      boxShadow: "2px 0 8px rgba(0,0,0,0.06)",
    }}>
      <div style={{ display: "flex", justifyContent: "space-between", padding: "6px 8px", borderBottom: "1px solid #e5e7eb", fontSize: 10, color: "#6b7280" }}>
        <span>{t(ZONE_LABELS[zoneId])}</span>
        <button onClick={onClose} style={closeBtn}>×</button>
      </div>
      <div style={{ padding: 6 }}>
        {zoneId === "context" && <ContextZone {...zoneProps} />}
        {zoneId === "tool"    && <ToolZone    {...zoneProps} />}
        {zoneId === "files"   && <FilesZone   {...zoneProps} />}
        {zoneId === "tray"    && <TrayZone    {...zoneProps} />}
        {zoneId === "mode"    && <ModeSlot    {...modeSlotProps} />}
      </div>
    </div>
  );
}
```

注：抽屉里的区组件复用现有的。它们已经在 Workbench 内部不再常驻渲染——改为按需出现在抽屉里。Workbench 原来渲染区的代码（`workbenchZones.filter().map`）删掉，区组件只通过抽屉渲染。

## 三、对话历史 + 导航条

### 导航条

```typescript
function TimelineBar({ messages }: { messages: WbMessage[] }) {
  // 按 5 分钟分组
  const groups: { label: string; msgIndex: number }[] = [];
  let lastTime = 0;
  for (let i = 0; i < messages.length; i++) {
    const bucket = Math.floor(i / 5) * 5; // 简化：每 5 条消息一组
    if (bucket !== lastTime) {
      groups.push({ label: `${i}`, msgIndex: i });
      lastTime = bucket;
    }
  }

  if (groups.length <= 1) return null; // 对话少于 5 条不显示

  return (
    <div style={{
      display: "flex", gap: 2, padding: "2px 8px",
      borderBottom: "1px solid #f3f4f6", overflowX: "auto",
    }}>
      {groups.map((g, i) => (
        <div key={i}
          onClick={() => { /* 滚动到 msgIndex */ }}
          style={{
            width: 4, height: 4, borderRadius: 2,
            background: "#3b82f6", cursor: "pointer",
            flexShrink: 0,
          }}
          title={`消息 #${g.msgIndex}`}
        />
      ))}
    </div>
  );
}
```

注：实际时间分组——在 `WbMessage` 里加 `timestamp: number` 字段（创建时自动填充 `Date.now()`）。按 `Math.floor((msg.timestamp - firstMsg.timestamp) / 300000)` 做 5 分钟分桶。
