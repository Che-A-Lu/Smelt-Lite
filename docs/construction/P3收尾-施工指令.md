## 一、快捷键补全

### 当前已有

| 快捷键 | 行为 |
|--------|------|
| Ctrl/Cmd+K | 搜索 |
| Ctrl/Cmd+N | 新建工作台 |
| Delete/Bksp | 删除选中卡片（多选弹确认） |
| Escape | 关闭面板/取消选中/关浮窗 |
| Space+拖拽 | 平移画布 |
| 滚轮 | 缩放 |
| Enter（工作台内） | 发送 |
| Shift+Enter（工作台内） | 换行 |

### 新增三个

**Space（无拖拽时）：快速预览。** 画布上有卡片被选中时，按 Space → 弹出浮层显示卡片详情（复用 CardView 的 `DetailView` 组件）。再按 Space 或 Escape 关闭。

Canvas 键盘事件中加：

```typescript
if (e.code === "Space" && !e.repeat && selectedIds.size === 1) {
  e.preventDefault();
  const cardId = [...selectedIds][0];
  setQuickPreview(cardId);
  return;
}
```

`quickPreview` state 处理：

```typescript
const [quickPreview, setQuickPreview] = useState<string | null>(null);

// 渲染：
{quickPreview && (
  <DetailView
    sphereId={index.cards[quickPreview]?.sphereId ?? ""}
    cardLabel={index.cards[quickPreview]?.label ?? ""}
    onClose={() => setQuickPreview(null)}
  />
)}
```

**Ctrl/Cmd+S：手动持久化。** 调 `flushAndWait()` ——触发 OPFS 立即写入。

Canvas 键盘事件中加：

```typescript
if ((e.ctrlKey || e.metaKey) && e.code === "KeyS") {
  e.preventDefault();
  flushAndWait().then(() => showToast("已保存"));
  return;
}
```

需要在 Canvas import 里加 `flushAndWait`。

**?（Shift+/）：快捷键帮助面板。** 工具栏加一个 `?` 按钮 → 弹出 `ShortcutHelp` 面板。

## 二、快捷键帮助面板

### 新建 `ui/panels/ShortcutHelp.tsx`

```typescript
export function ShortcutHelp({ onClose }: { onClose: () => void }) {
  const shortcuts = [
    { key: "Ctrl+K", desc: "搜索" },
    { key: "Ctrl+N", desc: "新建工作台" },
    { key: "Ctrl+S", desc: "手动保存" },
    { key: "Delete/Bksp", desc: "删除选中卡片" },
    { key: "Escape", desc: "关闭面板 / 取消选中" },
    { key: "Space", desc: "快速预览选中卡片" },
    { key: "Space+拖拽", desc: "平移画布" },
    { key: "滚轮", desc: "缩放画布" },
    { key: "Shift+框选", desc: "矩形框选" },
    { key: "Enter", desc: "发送消息" },
    { key: "Shift+Enter", desc: "换行" },
  ];

  return (
    <div onClick={(e) => { if (e.target === e.currentTarget) onClose(); }}
      style={{ position: "fixed", inset: 0, zIndex: 9999, display: "flex", alignItems: "center", justifyContent: "center", background: "rgba(0,0,0,0.06)" }}>
      <div onClick={(e) => e.stopPropagation()}
        style={{ width: 320, background: "#fff", border: "1px solid #d1d5db", padding: 16 }}>
        <div style={{ display: "flex", justifyContent: "space-between", alignItems: "center", marginBottom: 12 }}>
          <span style={{ fontSize: 13, fontWeight: 600 }}>键盘快捷键</span>
          <button onClick={onClose}
            style={{ border: "none", background: "none", fontSize: 14, cursor: "pointer", color: "#6b7280" }}>×</button>
        </div>
        {shortcuts.map((s) => (
          <div key={s.key} style={{ display: "flex", justifyContent: "space-between", padding: "3px 0", fontSize: 11, borderBottom: "1px solid #f3f4f6" }}>
            <span style={{ fontFamily: "monospace", color: "#3b82f6" }}>{s.key}</span>
            <span style={{ color: "#6b7280" }}>{s.desc}</span>
          </div>
        ))}
      </div>
    </div>
  );
}
```

### App.tsx 工具栏加 `?` 按钮

```typescript
const [showShortcuts, setShowShortcuts] = useState(false);

<button onClick={() => setShowShortcuts(true)} style={toolbarBtnStyle}>?</button>

{showShortcuts && <ShortcutHelp onClose={() => setShowShortcuts(false)} />}
```

## 三、i18n 遗漏补全

搜索全项目 `"` 字符串中硬编码的中文/英文提示，补 `t()` 调用。

已知遗漏（从之前检查记录）：

| 文件 | 位置 | 内容 |
|------|------|------|
| `CardView.tsx` | 右键菜单 | `"View detail"` → `t("card.viewDetail")` |
| `Workbench.tsx` | 暂停条 | `"Paused. Check results then continue."` → `t("wb.pauseHint")` |
| `Workbench.tsx` | 继续按钮 | "继续" → `t("wb.continueBtn")` |
| `ExportPanel.tsx` | 标签 | `"Card name"`、`"Author"`、`"Version"`、`"Tags"` → 对应 t() |
| `ExportPanel.tsx` | 密码强度 | `"Strength: "` → `t("export.pwStrength")` |

在 `foundation/i18n.ts` 补充缺少的 key，每个 key 有 zh + en 两列。

批量检查方法：`grep -rn "style=" src/ | grep -v "t("` 找内联的硬编码字符串。但最有效的做法是跟着上面表格逐一改。
