## 一、主题卡应用

### 现状

`InteractionConfig` 已定义全套参数，但值从 `defaultInteraction()` 读取后直接用了——没有覆盖入口。

### 改动

**1. 拖入卡座 → 检测是否主题卡**

Dock 的 `activateCardInDock` 中（或 Canvas 的 `onCardDragEnd` 落点判定中），加检测：

```typescript
// 卡片拖入卡座前：检查是否包含 interaction 字段
if (card.sphereId) {
  const files = await readSphereFiles(card.sphereId);
  const text = files.map((f) => f.content).join("\n");
  try {
    const json = JSON.parse(text);
    if (json.interaction && typeof json.interaction === "object") {
      // 是主题卡 → 覆盖全局交互参数
      activateTheme(json.interaction);
    }
  } catch { /* 不是 JSON，正常激活 */ }
}
```

**2. Canvas 接受主题覆盖**

```typescript
// Canvas 中：
const [themeOverride, setThemeOverride] = useState<Partial<InteractionConfig> | null>(null);

// 合并默认 + 覆盖
const effectiveInteraction = themeOverride
  ? deepMerge(defaultInteraction(), themeOverride)
  : interaction;
```

`deepMerge` 浅层合并即可——`interaction` 只有两层深。

**3. 从卡座拖出 → 恢复默认**

```typescript
// Dock 拖出时：
if (wasThemeCard) setThemeOverride(null);
```

## 二、小地图

### 新建 `ui/canvas/Minimap.tsx`

```typescript
interface MinimapProps {
  cards: CardEntry[];
  view: { x: number; y: number; zoom: number };
  onNavigate: (cx: number, cy: number) => void;
}
```

**渲染逻辑：**

- 200×150px 缩略图，右下角固定
- 背景色略深于画布底色（#f0f1f3）
- 每张卡片渲染为一个 2×3px 色块（仿扑克牌比例）——颜色取自 `fileCategory(card.label).color`
- 当前视口渲染为半透明蓝色矩形
- 点击小地图 → 计算对应位置 → `onNavigate(-cx, -cy)`

**卡片坐标映射：**

```typescript
// 找出所有卡片的边界
const minX = Math.min(...cards.map((c) => c.position.x));
const maxX = Math.max(...cards.map((c) => c.position.x));
const minY = Math.min(...cards.map((c) => c.position.y));
const maxY = Math.max(...cards.map((c) => c.position.y));

// 映射到 200×150
const mapX = (x: number) => ((x - minX) / (maxX - minX)) * 180 + 10;
const mapY = (y: number) => ((y - minY) / (maxY - minY)) * 130 + 10;
```

**边缘情况**：卡片少于 3 张时不渲染——小地图无意义。

### Canvas 集成

```typescript
// Canvas 中：
const showMinimap = cards.length >= 3;

{showMinimap && (
  <Minimap
    cards={cards}
    view={view.current}
    onNavigate={(cx, cy) => setView({ x: -cx, y: -cy, scale: view.current.zoom })}
  />
)}
```

## 三、右键菜单组件化

### 新建 `ui/components/ContextMenu.tsx`

当前右键菜单是 `document.createElement("div")` + `innerHTML` + 手动事件绑定。分散在 CardView、ContextZone、HistoryZone、ToolZone、PackView。

统一替换为 React 组件：

```typescript
interface ContextMenuItem {
  label: string;
  danger?: boolean;    // 红色文字
  separator?: boolean;  // 上方加分割线
  onClick: () => void;
}

interface ContextMenuProps {
  x: number; y: number;
  items: ContextMenuItem[];
  onClose: () => void;
}

export function ContextMenu({ x, y, items, onClose }: ContextMenuProps) {
  useEffect(() => {
    const close = (e: MouseEvent) => {
      // 点击菜单外 → 关闭
      onClose();
    };
    // 延迟绑定，防止触发右键的 click 事件立即关闭菜单
    setTimeout(() => document.addEventListener("click", close), 0);
    return () => document.removeEventListener("click", close);
  }, [onClose]);

  return (
    <div style={{ position: "fixed", left: x, top: y, background: "#fff", border: "1px solid #d1d5db", padding: "4px 0", zIndex: 9999, fontSize: 11, minWidth: 100 }}>
      {items.map((item, i) => (
        <React.Fragment key={i}>
          {item.separator && <div style={{ borderTop: "1px solid #e5e7eb", margin: "2px 0" }} />}
          <div
            onClick={() => { item.onClick(); onClose(); }}
            style={{ padding: "4px 12px", cursor: "pointer", color: item.danger ? "#ef4444" : "#1a1a2e" }}
          >
            {item.label}
          </div>
        </React.Fragment>
      ))}
    </div>
  );
}
```

### 替换五个地方的右键菜单

**CardView（约第 240-262 行）：** 删掉 `onContextMenu` 里的 `document.createElement` 逻辑 → 改为 setState 传菜单项给 `<ContextMenu>`。

```typescript
const [ctxMenu, setCtxMenu] = useState<{ x: number; y: number } | null>(null);

const onContextMenu = (e: React.MouseEvent) => {
  e.preventDefault();
  setCtxMenu({ x: e.clientX, y: e.clientY });
};

// 渲染：
{ctxMenu && (
  <ContextMenu x={ctxMenu.x} y={ctxMenu.y} items={[
    { label: "查看详情", onClick: () => setShowDetail(true) },
    { label: "导出文件", onClick: handleExportFile },
    { label: "打包为 .card", onClick: () => onExportCard?.(card.id), separator: true },
  ]} onClose={() => setCtxMenu(null)} />
)}
```

**ContextZone（约第 44-60 行）：** 同样改法。`onContextMenu` → setState → 渲染 `<ContextMenu>`。菜单项：`t("wb.contextEdit")` + `t("ui.delete")`。

**HistoryZone（约第 21-40 行）：** 同样改法。菜单项：`t("wb.extractCard")`。

**ToolZone：** 同样改法。菜单项：`t("ui.delete")`。

**PackView：** 同样改法。菜单项：`t("pack.rename")` + `t("pack.delete")`。

全部删掉 `document.createElement`、`innerHTML`、`querySelector`、手动 `addEventListener`。
