# 卡片快照 · 施工指令

## 目标

每张卡片一张小缩略图 PNG，存在球目录里。CardView 只渲染 `<img>`，不需要知道文件类型。LOD 四级：远了用小的，近了换大的，AI 临时文件不做。

---

## 一、文件类型覆盖

| 类型 | 扩展名 | 如何生成快照 | 依赖 |
|------|--------|-------------|------|
| 图片 | png/jpg/jpeg/gif/webp/svg | `<img>` 加载原文件 → Canvas 缩放到目标尺寸 → 导出 PNG | 零 |
| 文本 | md/txt/json/csv/ts/js/tsx/jsx/py/go/rs/yaml/xml/css/html | `<pre>` 渲染格式化文本 → Canvas 绘制 → 导出 PNG | 零 |
| CSV | csv | 解析前 20 行 → Canvas 画迷你表格（表头加粗、列宽自适应） | 零 |
| 其他 | xlsx/docx/pdf/zip/mp3/mp4 等 | 不生成快照 | — |

## 二、快照生成器

新建 `features/snapshot.ts`：

### 2.1 统一入口

```typescript
export async function generateSnapshot(
  file: File | { name: string; data: ArrayBuffer },
  size: { w: number; h: number },
): Promise<Blob | null> {
  const ext = file.name.toLowerCase().slice(file.name.lastIndexOf("."));
  
  if (IMAGE_EXTS.has(ext)) return snapshotImage(file, size);
  if (TEXT_EXTS.has(ext)) return snapshotText(file, size);
  if (ext === ".csv") return snapshotCSV(file, size);
  
  return null; // 不支持的类型
}
```

### 2.2 图片快照

```typescript
async function snapshotImage(file, size): Promise<Blob | null> {
  const blob = file instanceof File ? file : new Blob([file.data]);
  const url = URL.createObjectURL(blob);
  const img = new Image();
  img.src = url;
  await img.decode();
  
  const canvas = document.createElement("canvas");
  canvas.width = size.w;
  canvas.height = size.h;
  const ctx = canvas.getContext("2d")!;
  
  // 缩小填满（cover 裁切）
  const scale = Math.max(size.w / img.width, size.h / img.height);
  const dw = img.width * scale;
  const dh = img.height * scale;
  const dx = (size.w - dw) / 2;
  const dy = (size.h - dh) / 2;
  ctx.drawImage(img, dx, dy, dw, dh);
  
  URL.revokeObjectURL(url);
  
  return new Promise((resolve) => {
    canvas.toBlob((b) => resolve(b), "image/png", 0.7);
  });
}
```

### 2.3 文本快照

```typescript
async function snapshotText(file, size): Promise<Blob | null> {
  const text = file instanceof File
    ? await file.text()
    : new TextDecoder().decode(file.data);
  
  const lines = text.split("\n").slice(0, 30);
  
  // 用离屏 DOM 渲染文本
  const pre = document.createElement("pre");
  pre.textContent = lines.join("\n");
  pre.style.cssText = `
    font-family: monospace; font-size: 11px; line-height: 1.4;
    color: #1a1a2e; background: #fff; padding: 8px;
    width: ${size.w * 2}px; white-space: pre-wrap; word-break: break-all;
  `;
  document.body.appendChild(pre);
  
  // html2canvas 或手动 Canvas 绘制
  const canvas = document.createElement("canvas");
  canvas.width = size.w;
  canvas.height = size.h;
  const ctx = canvas.getContext("2d")!;
  ctx.fillStyle = "#ffffff";
  ctx.fillRect(0, 0, size.w, size.h);
  ctx.fillStyle = "#1a1a2e";
  ctx.font = "9px monospace";
  
  for (let i = 0; i < Math.min(lines.length, 20); i++) {
    const line = lines[i].slice(0, 60);
    ctx.fillText(line, 4, 14 + i * 13);
  }
  
  document.body.removeChild(pre);
  
  return new Promise((resolve) => {
    canvas.toBlob((b) => resolve(b), "image/png", 0.6);
  });
}
```

实际实现时文本渲染可以用更简单的方法——直接用 Canvas `fillText`，不需要 HTML 渲染。上面给出了两种方向。

### 2.4 CSV 快照

类似文本，但第一行表头加粗，列宽自适应，画迷你表格框线。

## 三、快照存储

### 3.1 修改 `createSphereFromFile`

文件导入时，顺便生成 L1 快照（300×400）存到球目录：

```typescript
// 在 createSphereFromFile 末尾加
if (shouldSnapshot(file.name)) {
  const snap = await generateSnapshot(file, { w: 300, h: 400 });
  if (snap) {
    const snapHandle = await sphereDir.getFileHandle("_snapshot.png", { create: true });
    const w = await snapHandle.createWritable();
    await w.write(await snap.arrayBuffer());
    await w.close();
  }
}

// AI 临时文件判断
function shouldSnapshot(fileName: string): boolean {
  // AI 临时文件特征：在工作台 File 区，尚未"建卡"
  // createSphereFromFile 是被用户从桌面拖入的文件 → 全做快照
  // AI 产出的临时文件走的是 createCard（无球）→ 不做快照
  return true;
}
```

**关键区分**：用户从桌面拖入的文件 → `createSphereFromFile` → 做快照。AI 产出的临时文件 → `createCard`（无 sphere，在 File 区未建卡）→ 不做快照。

### 3.2 新增 `snapshotExists` 和 `readSnapshot`

在 `storage.ts` 中加：

```typescript
export async function readSnapshot(sphereId: string): Promise<string | null> {
  const dir = await getRoot();
  const spheresDir = await dir.getDirectoryHandle("spheres");
  const sphereDir = await spheresDir.getDirectoryHandle(sphereId);
  
  try {
    const handle = await sphereDir.getFileHandle("_snapshot.png");
    const file = await handle.getFile();
    return URL.createObjectURL(file);
  } catch {
    return null; // 没有快照
  }
}
```

### 3.3 文件修改时更新快照

`updateCard` 如果检测到卡片内容变化 → `generateSnapshot({w: 300, h: 400})` → 覆盖 `_snapshot.png`。

## 四、CardView 改预览区

### 4.1 状态

```typescript
const [snapshotUrl, setSnapshotUrl] = useState<string | null>(null);
const [snapshotLoading, setSnapshotLoading] = useState(false);
const snapshotCache = useRef(new Map<string, string>()); // 全局 LRU
```

### 4.2 IntersectionObserver 控制加载

```typescript
useEffect(() => {
  if (!sphereId) return;
  const el = elRef.current;
  if (!el) return;

  const cached = snapshotCache.current.get(sphereId);
  if (cached) { setSnapshotUrl(cached); return; }

  const observer = new IntersectionObserver(
    ([entry]) => {
      if (entry.isIntersecting) {
        setSnapshotLoading(true);
        readSnapshot(sphereId).then((url) => {
          if (url) {
            snapshotCache.current.set(sphereId, url);
            setSnapshotUrl(url);
          }
          setSnapshotLoading(false);
        });
        observer.disconnect();
      }
    },
    { rootMargin: "200px" } // 提前 200px 加载
  );
  observer.observe(el);
  return () => observer.disconnect();
}, [sphereId]);
```

### 4.3 预览区渲染

```typescript
<div style={{ position: "absolute", left: 8, top: 28, right: 8, bottom: 24 }}>
  {snapshotUrl ? (
    <img src={snapshotUrl} style={{ width: "100%", height: "100%", objectFit: "contain", borderRadius: 2 }} />
  ) : snapshotLoading ? (
    <div style={{ fontSize: 9, color: "#d1d5db" }}>...</div>
  ) : sphereId ? (
    // 有球但没快照（其他文件类型）→ 显示类型图标 + 文件名
    <div style={{ display: "flex", flexDirection: "column", alignItems: "center", justifyContent: "center", height: "100%" }}>
      <span style={{ fontSize: 20, opacity: 0.3 }}>{category.label}</span>
    </div>
  ) : (
    // 无球（AI 临时文件）→ 空白
    null
  )}
</div>
```

底部留 24px 给标签区（8px 上 + 标签行）。

### 4.4 标签移到底部

当前标签在预览区但只占了第一行。改为：

```
预览区（28px ~ 卡高-24px）  ← 缩略图填满
标签区（卡高-24px ~ 卡高-8px） ← 标签缩略显示，9px 灰色
```

### 5.1 右键菜单加"查看详情"

```typescript

  const files = await readSphereFiles(card.sphereId);
  // 如果原文件是图片，直接显示原图
  // 如果是文本，Canvas 渲染 600×800 文本 → 展示
  // 显示在浮层中
}
```

浮层中 `<img>` 渲染 600×800 快照，背景半透明遮罩，点击遮罩关闭。

### 5.2 Canvas 缩放联动

CardView 监听 Canvas 的 `view.scale`：

- scale < 1.5 → 渲染 L1 快照（300×400）
- 1.5 ≤ scale < 3 → 渲染 L2 快照（600×800，按需生成）
- scale ≥ 3 → 直接用 Canvas fillText 渲染原文（无限清晰，不存文件）

```typescript
// CardView 新增 prop: viewScale
useEffect(() => {
  if (!sphereId) return;
  if (viewScale >= 3) {
    // L3：读原文 → Canvas 直接渲染文本
    renderTextDirectly(sphereId, viewScale);
  } else if (viewScale >= 1.5) {
    // L2：生成 600×800 快照（如果还没有）
    loadSnapshot(sphereId, 600, 800);
  } else {
    // L1：默认快照
    loadSnapshot(sphereId, 300, 400);
  }
}, [viewScale]);
```

## 六、LRU 缓存

缩略图 blob URL 缓存：

```typescript
const snapshotCache = new Map<string, string>();
const MAX_CACHE = 80; // 80 张快照 = ~500KB-1MB

function cacheSnapshot(sphereId: string, url: string) {
  if (snapshotCache.size >= MAX_CACHE) {
    const first = snapshotCache.keys().next().value;
    const old = snapshotCache.get(first);
    if (old) URL.revokeObjectURL(old);
    snapshotCache.delete(first);
  }
  snapshotCache.set(sphereId, url);
}
```

## 八、验收

1. `npx tsc --noEmit` 零错误
2. 从桌面拖入一张图片 → 卡片出现 → 预览区显示缩略图
3. 从桌面拖入一个 md/txt → 卡片出现 → 预览区显示文本渲染缩略图
4. 从桌面拖入一个 CSV → 卡片出现 → 预览区显示迷你表格
5. 拖入 xlsx/pdf → 预览区显示大号类型图标（不崩溃）
6. AI 回复后 File 区出现临时文件 → 临时文件卡片不做快照（预览区空白或只显示类型图标）
7. 卡片滚出视口 → 不加载快照。滚回来 → 从缓存取，不重新生成
8. Canvas 缩放到 2x → 卡片预览换大图（不模糊）
9. 右键"查看详情" → 弹出大号浮层
10. 球目录里存在 `_snapshot.png` 文件
