## 工程师那边

另一个对话是工程师角色。他只写代码，不该设计。你把施工指令放到 `.card概念/` 文件夹，Dalu 会把指令转给他。

工程师写完 → Dalu 会说"他写好了" → 你需要**检查代码是否正确实现**：
1. 用 PowerShell 把 `.card概念/app/` 同步到 `smelt-github/app/`
2. 跑 `npx tsc --noEmit` 确认零错误
3. 逐条对照施工指令验收
4. 向 Dalu 报告检查结果
5. 如果通过，`git push` 到 GitHub

**同步命令（从 .card概念 → smelt-github）：**
```bash
powershell -Command "Copy-Item -Path 'E:\My agent file\My file\.card概念\app\src' -Destination 'E:\My agent file\My file\smelt-github\app\src' -Recurse -Force"
```

**推 GitHub（需要 VPN 开着，代理端口 7897）：**
```bash
cd "E:\My agent file\My file\smelt-github"
git add -A
git commit -m "描述改动"
git push
```

**反向同步（smelt-github → .card概念，如果代码在那边改的）：**
```bash
powershell -Command "Copy-Item -Path 'E:\My agent file\My file\smelt-github\app\src' -Destination 'E:\My agent file\My file\.card概念\app\src' -Recurse -Force"
```

**强制清 OPFS 缓存（浏览器 Console）：**
```javascript
(async()=>{const r=await navigator.storage.getDirectory();for await(const[n]of r){try{await r.removeEntry(n,{recursive:true})}catch(e){}};alert("已清除");location.reload()})()
```
