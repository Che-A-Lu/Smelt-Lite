# .card Standard Prompts / .card 标准提示词

> These prompts are the entry point to the `.card` format. Paste them into any AI tool (ChatGPT, Claude, Kimi, DeepSeek, etc.) and the AI will output structured JSON ready for Smelt Lite.
>
> 这些提示词是 `.card` 格式的入口。粘贴到任何 AI 工具中，AI 会输出结构化 JSON，直接拖入 Smelt Lite 打包。

## Quick start / 快速开始

1. Copy a prompt below / 复制下方提示词
2. Paste into your AI conversation / 粘贴到 AI 对话中
3. AI outputs JSON → drag into [Smelt Lite](https://che-a-lu.github.io/Smelt-Lite/) / 拖入 Smelt Lite
4. Download your `.card` / 下载 .card

## What's here / 内容

- **[提示词.md](提示词.md)** — Full prompt library with three scenarios (pre-install, post-chat, recovery) and three output modes (decision record, work log, quick note). / 完整提示词库，三种场景 + 三种输出模式。
